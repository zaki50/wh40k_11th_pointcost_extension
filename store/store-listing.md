## 概要（short description / 最大132文字）

```
wikiwiki.jp の 40k（11版）ユニットデータページに、公式ポイントマニュアルのポイントコストを自動で挿入表示。モデル数別・編入順の値もまとめて確認できます。
```

## 詳細な説明（detailed description / 最大16,000文字）

```
「ウォーハンマー40,000（11版）日本語プレイヤーwiki」(wikiwiki.jp/40k-11th) の
ユニットデータページには、そのユニットのポイントコストが載っていません。
この拡張機能は、閲覧中のページのアーミー名・ユニット名を読み取り、公式の
Munitorum Field Manual（mfm.warhammer-community.com）から対応するポイントを取得して、
ページ上部に見やすいカードで自動挿入します。wiki を見ながらリスト構築ができます。

■ 主な機能
・ユニットデータページを開くだけで、ポイントコストを自動表示（操作不要）
・モデル数別のポイント（例: 5体 / 10体）をすべて表示
・ロスターへの編入順によるポイント（1個目〜2個目 / 3個目 など）も表示
・日本語版の表記と英語名を併記（正体が分かりやすい）
・取得結果は一時的にキャッシュし、素早く表示

■ 対応アーミー
帝国（インペリウム）／渾沌（ケイオス）／異種族（ゼノ）の主要 28 アーミーに対応。

■ 使い方
1. 拡張機能をインストール
2. wikiwiki.jp/40k-11th の「アーミー/〇〇/ユニットデータ/△△」ページを開く
3. ページ上部にポイントコストのカードが表示されます
（該当ページが未作成でも、URL が合っていれば表示されます）

■ プライバシー
・個人情報の収集・送信は一切行いません。
・ポイント取得のため mfm.warhammer-community.com へアクセスし、その結果をブラウザ内に
　一時キャッシュするためだけに storage を使用します。
・アクセス先は上記サイトと、動作対象の wikiwiki.jp/40k-11th のみです。

■ 免責事項
本拡張機能はファンによる非公式ツールであり、Games Workshop 社とは一切関係ありません。
「Warhammer 40,000」等の名称・データは Games Workshop 社の著作物・商標です。
ポイント値は公式サイトの内容に依存し、正確性・最新性を保証するものではありません。

■ オープンソース
ソースコードは Apache License 2.0 で公開しています。
https://github.com/zaki50/wh40k_11th_pointcost_extension
```

## プライバシー実務（審査フォームの記入用）

### 単一用途の説明（Single purpose）

```
閲覧中の wikiwiki.jp/40k-11th ユニットデータページに対し、対応するユニットのポイントコストを
公式 Munitorum Field Manual から取得してページ内に表示する、という単一機能に限定しています。
```

### 権限の正当化（Permission justification）

- **storage**
  ```
  取得したポイント情報を一定時間ブラウザ内にキャッシュし、再取得を減らして高速表示するために使用します。個人データは保存しません。
  ```
- **ホスト権限 `https://mfm.warhammer-community.com/*`**
  ```
  ユニットのポイントコストは公式の Munitorum Field Manual（このドメイン）に掲載されているため、当該ページを取得・解析する目的でのみアクセスします。
  ```
- **リモートコードの使用**: なし（すべてのコードは拡張機能に同梱）
- **データ収集**: なし（ユーザーデータの収集・使用・送信はいずれも行わない）
